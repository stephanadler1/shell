dev(1), src(1) Manual Page

Name

Various commands to switch the current directory to a working copy of a repository:

  • dev
  • src

Synopsis

dev [working_copy_folder_name]

src [working_copy_folder_name]

Description

Can be used to quickly switch between different working copy folders (enlistments). The command switches to the new folder and updates the window title accordingly. It does set any environment variables specific to the working copy folder.

Environment

SOURCES_ROOT
If defined it is used to determine the root folder for working copies.
