random(1) Manual Page

Name

random – Generates cryptographically strong random data of the requested size.

Synopsis

random number_of_bytes

Description

Generates cryptographically strong random data using a PRNG. This can be used to generate machine keys for the machineKey in IIS.

By default it uses 1,000,000 iterations to generate the random data.
