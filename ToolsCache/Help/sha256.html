md5(1), sha1(1), sha256(1), sha512(1) Manual Page

Name

Various command calculating file hashes:

  • md5 – Calculates the MD5 hash value for the file.
  • sha1 – Calculates the SHA-1 128 bit hash value for the file.
  • sha256 – Calculates the SHA-2 256 bit hash value for the file.
  • sha512 – Calculates the SHA-2 512 bit hash value for the file.

Synopsis

md5 File

sha1 File

sha256 File [ExpectedHash]

sha512 File [ExpectedHash]

Description

Calculates the specified hash value for the File. For sha256 and sha512 when a ExpectedHash is specified it is compared to the calculated hash value.
