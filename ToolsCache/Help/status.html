status(1), toggle-status(1) Manual Page

Name

  • status – sets the color or status of an attached presence light device.
  • toggle-status – toggles the status of an attached presence light device between red (busy) and green (available).

Synopsis

status (blue|cyan|green|magenta|orange|red|white|yellow|available|busy|dnd|away|incoming)

status (off|on)

toggle-status

Description

Both commands either set the presence light device to the specified color or to the color associated with the specified status.

  • available – green
  • busy – red
  • away – yellow
  • dnd – magenta for do not disturb.
  • incoming – blue for incoming call or incoming IM message.

The command status off turns the presence light off.

The command status on returns the presence light to the last status set through toggle-status.

The commands support the following presence light devices:

  • Embrava Blynclight. In order to interact with the device the Embrava Connect software is required. It can be downloaded through the link below.
  • PLENOM Kuando Busylight. (untested)

Files

toggle-status uses a semaphore file in %TEMP%\presence-light-busy.txt to keep track of the current status. If the light is updated by other applications the state of the file might not match the actual state of the light. Using toggle-status multiple times (at most 2 times) will bring the light back into the desired state.

In addition to the the command line tool, a shortcut named "Update Status" has been added to the desktop, which will call toggle-status to update the status of the light.
