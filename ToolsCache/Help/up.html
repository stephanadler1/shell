..(1), ...(1), ....(1), .....(1) Manual Page

Name

Various commands for folder/directory navigation:

  • .. – Navigates to the parent of the current folder. One level up.
  • ... – Navigates two levels up.
  • .... – Navigates three levels up.
  • ..... – Navigates four levels up.

Synopsis

.. [subdirectory to navigate into from the new level]

... [subdirectory to navigate into from the new level]

.... [subdirectory to navigate into from the new level]

..... [subdirectory to navigate into from the new level]

Description

Navigates the number of directory levels up the hierarchy and from there navigates down the folders in the optional parameter.
