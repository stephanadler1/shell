toggle-msbuildlogging(1) Manual Page

Name

toggle-msbuildlogging – Toggles between MSBuild file logging on and off.

Synopsis

toggle-msbuildlogging

Description

Logging is a feature of the build customization in b(1), bcc(1), bdiag(1), msb(1), nr(1), slngen(1). See there for more details.

Environment

MSBUILD_ENABLELOGGING
If defined enables the MSBuild file logging of diagnostics, warnings and errors.
