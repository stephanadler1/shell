git(1) Manual Page

Name

Various git aliases added for the developer shell.

  • git br – Shortcut for git branch.
  • git cl – Shortcut for git clean.
  • git co – Shortcut for git checkout.
  • git cod – Shortcut for git checkout develop.
  • git com – Shortcut for git checkout master or the any basename set as origin/HEAD, like main.
  • git db – Reveals the name of the default branch of the repository.
  • git delete – Shortcut to delete a branch incl. from origin.
  • git gh – Get the head commit of the current branch and copy it into the clipboard.
  • git hist – Gets the history of the current branch.
  • git lol – Brief version of git log.
  • git nb – Creates a new development branch and sets the upstream.
  • git nf – Creates a new feature branch and sets the upstream.
  • git nr – Removes a fully merged development branch locally and remote.
  • git pause – Creates a save point commit in the current branch to save the current work. Similar to a stash.
  • git pf – force pushes with lease.
  • git rbd – Shortcut for git rebase origin/develop.
  • git rbm – Shortcut for git rebase origin/master or the any basename set as origin/HEAD, like main.
  • git remove – Shortcut to force delete a branch incl. from origin.
  • git rhd – Shortcut for git reset --hard origin/develop.
  • git rhm – Shortcut for git reset --hard origin/master or the any basename set as origin/HEAD, like main.
  • git st – Shortcut for git status.
  • git start – Removes the head commit (assuming it is a save point) to resume the work.
  • git up – Updates the current branch from its remote incl. submodules.
  • git wipe – Removes a save point.

Run git config --get-regexp alias. | sort to see the full list of local and global aliases defined for Git.
