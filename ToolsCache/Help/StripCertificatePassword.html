StripCertificatePassword(1) Manual Page

Name

StripCertificatePassword – Removes the password protection from a X.509 certificate file containing the private key of the certificate.

Synopsis

StripCertificatePassword CertificateFile Password [OutputFile]

Description

The script loads the CertificateFile and opens the password protected certificate with the given Password. It then stores the certificate without password protection in OutputFile. If no OutputFile is given the default value is used with CertificateFile.nopassword.pfx.
