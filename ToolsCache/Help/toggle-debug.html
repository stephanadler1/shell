toggle-debug(1) Manual Page

Name

toggle-debug – Toggles between debug logging on and off.

Synopsis

toggle-debug

Description

Debug logging in all scripts is controlled by the environment variable _DEBUG being set (logging is on) and not set (logging is off). This command switches between the two states such that subsequent commands emit logging.

Environment

_DEBUG
If defined enables diagnostic output of shell scripts. All shell scripts should have @if not defined _DEBUG echo off as their first line.
