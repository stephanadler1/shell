toggle-msbuildisolation(1) Manual Page

Name

toggle-msbuildisolation – Toggles between MSBuild build isolation on and off.

Synopsis

toggle-msbuildisolation

Description

Build isolation is a feature of the build customization in b(1), bcc(1), bdiag(1), msb(1), nr(1), slngen(1). See there for more details.

Environment

MSBUILD_DISABLEISOLATION
If defined disables the build isolation.
