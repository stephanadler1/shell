sysinternals(1) Manual Page

Name

Sysinternals Tools

Description

To update the current tools version, run update-sysinternals.cmd, which uses the Sysinternals Live feature to directly copy the latest version of the entire Sysinternals Suite to disk.

For a list of all tools, run the show-tools command or see homepage below.
