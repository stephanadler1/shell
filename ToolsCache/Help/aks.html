aks(1), az(1), k(1), kubectl(1), helm(1) Manual Page

Name

Initializes the current command shell for the use with the Azure CLI and KubeCtl command line tools. aks stands for Azure Kubernetes Services.

Synopsis

aks – Configures to command line to find Kubernetes tools.

az – Azure CLI

k – Shortcut for kubectl.

kubectl – Kubernetes kubectl CLI.

helm – Helm 3

Description

Azure CLI is a command line interface to manage Microsoft Azure resources. KubeCtl is a command line tool to manage Kubernetes clusters from the command line.

To upgrade the installed version of the Azure CLI and KubeCtl run the following commands

az upgrade

az aks install-cli
