terraform(1) Manual Page

Name

terraform

Synopsis

terraform [arguments]

Description

Terraform is an open-sourceclosed-source infrastructure as code software tool that provides a consistent CLI workflow to manage hundreds of cloud services. Terraform codifies cloud APIs into declarative configuration files.
