ac(1), browse(1), rg(1)

Name

Various commands to for working with repositories in addition to the commands implemented by the basic repository tools.

  • ac – short for all changes, accesses the history of the current branch of the repository.
  • browse – opens a repository browser.
  • rg – short for revision graph, shows the revision graph for the repository and the current branch.

Synopsis

ac

browse

rg

Description

In order to use any of these commands you have to install additional software on your system. See the links in the see also section below.
